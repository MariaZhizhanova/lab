### Выведите в консоль булевую переменную False, не используя слово False в строке или изначально присвоенную булевую переменную. Программа должна занимать не более двух строк редактора кода.

```python
one = "1"
print(bool())
```

![image](https://github.com/MariaZhizhanova/lab/assets/145640698/8156ede5-56c2-4174-80a5-3a03d18eb3e2)


- В данном коде выводятся две строки с использованием функции `print()`. Каждая строка содержит разные значения:

1. `one = "1"`:  В этой строке создается переменная one, и ей присваивается значение "1". Здесь "1" - это строка, так как она заключена в двойные кавычки.
2. `print(bool())`: используется для хранения логических значений True или False.: В этой строке вызывается функция bool() без аргументов. Функция bool() используется для определения булевого значения переменных или выражений. Если аргумент не передан, она по умолчанию возвращает False. Затем результат, который в данном случае равен False, выводится на экран с помощью функции print().

Итак, этот код создает переменную one с значением "1", но не использует ее, и затем выводит False на экран, так как вызов bool() без аргументов возвращает False.

## Самостоятельная работа №5
### Создайте три переменные: день (тип данных - числовой), месяц (тип данных - строка), год (тип данных - числовой) и выведите в консоль текущую дату в формате: “Сегодня день месяц год. Всего хорошего!” используя F строку и оператор end внутри print(), в котором вы должны написать фразу “Всего хорошего!”. Программа должна занимать не более двух строк редактора кода.

```python
day, month, year = 24, "сентября", 2023
print(f"Сегодня {day} {month} {year} ", end="Всего хорошего!")
```
![image](https://github.com/MariaZhizhanova/lab/assets/145640698/b6d54c1c-5e7c-406c-b422-7fae810c9101)

- В данном коде выводятся две строки с использованием функции `print()`. Каждая строка содержит разные значения:

1. `day, month, year = 24, "сентября", 2023`: В этой строке кода создаются три переменные day, month и year, и им присваиваются значения: 24, "сентября" и 2023 соответственно. Таким образом, у вас есть переменные с информацией о дне, месяце и годе.
2. `print(f"Сегодня {day} {month} {year} ", end="Всего хорошего!")`:В этой строке кода используется функция print() для вывода текста на экран. Строка внутри print() начинается с буквы "Сегодня", а затем через фигурные скобки {} вставляются значения переменных day, month и year, которые были объявлены ранее. end="Всего хорошего!" указывает, что после вывода этой строки текст "Всего хорошего!" будет добавлен без перевода строки.

## Самостоятельная работа №8
### Переведите предложение ‘HELLO WORLD’ в нижний регистр. Программа должна занимать не более двух строк редактора кода.
```python
word = "HELLO WORLD"
print(word.lower())
```
- ![image](https://github.com/MariaZhizhanova/lab/assets/145640698/bfff45d9-4725-4ddf-85ff-e5498e7a3376)

- Функция lower() преобразует все символы в строке в нижний регистр
- word = "HELLO WORLD" - В этой строке создается переменная word, и в нее присваивается строковое значение "HELLO WORLD". То есть, мы создали строку, состоящую из заглавных букв.

print(word.lower()) - Здесь мы применяем метод .lower() к строке, хранящейся в переменной word. Метод .lower() используется для преобразования всех букв в строке в нижний регистр. После применения этого метода, строка "HELLO WORLD" будет выглядеть как "hello world".

Итак, в результате выполнения этого кода, он выведет "hello world", преобразовав все буквы из верхнего регистра в нижний с помощью метода .lower().
