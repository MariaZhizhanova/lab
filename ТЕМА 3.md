## Самостоятельная работа №1
### Напишите программу, которая преобразует 1 в 31.
### Для выполнения поставленной задачи необходимо обязательно и только один раз использовать:
### -	Цикл for
### -	*= 5
### -	+= 1
### Никаких других действий или циклов использовать нельзя.

```python
for a in range(1, 7):
    a*=5
    a+=1
print(a)
```
![image](https://github.com/MariaZhizhanova/lab/assets/145640698/3044f7db-34c7-4705-8d21-01ba9f1d477c)

### Вывод
  Данный код является циклом for и выполняет следующие действия:

1. Создает переменную a и устанавливает ее значение равным 1.
2. Запускает цикл for с диапазоном от 1 до 7 (исключая 7).
3. На каждой итерации цикла:

- Умножает значение a на 5: a*=5 (эквивалентно a = a * 5).

- Прибавляет 1 к a: a+=1 (эквивалентно a = a + 1).
4. После завершения цикла, выводит значение переменной a.

Таким образом, код последовательно умножает a на 5 и прибавляет 1 на каждой итерации цикла, после чего выводит окончательное значение a. В данном случае, вывод будет равен 78126.
